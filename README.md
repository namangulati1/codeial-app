# Codeial Social Media Web App

Codeial is a social media web application built using React. It allows users to create an account, log in, view and post content, and interact with other users through comments and likes.

## Features

- User authentication: Sign up and log in to the application.
- View posts: Browse through a feed of posts from other users.
- Create posts: Share your thoughts and ideas by creating your own posts.
- Like and comment: Interact with posts by liking and commenting on them.
- User profiles: View user profiles to see their posts and information.
- Search users: Search for other users by name or username.

## Installation

1. Clone the repository: `git clone https://github.com/namangulati1/codeial-app`
2. Navigate to the project directory: `cd codeial-react-app`
3. Install the dependencies: `npm install`

## Usage

1. Start the development server: `npm start`
2. Open your browser and visit `http://localhost:3000` to access the Codeial web app.

## Technologies Used

- React
- React Router
- React Context API
- React Hooks
- HTML/CSS
- JavaScript

## Folder Structure

The project structure is organized as follows:

- `src`: Contains the application source code.
  - `components`: Reusable components used in the app.
  - `pages`: Individual pages/screens of the app.
  - `api`: API functions for interacting with the backend.
  - `hooks`: Custom hooks used in the app.
  - `styles`: Stylesheets for styling the components.
  - `App.js`: Main component that renders the app.
  - `index.js`: Entry point of the app.
